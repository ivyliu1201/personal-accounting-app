# FLOW.md

## 1. 首頁主要流程

```mermaid
flowchart TD
A[進入首頁] --> B{是否已登入}
B -- 否 --> C[顯示首頁版型但不可操作]
C --> D[點登入按鈕]
C --> E[點首頁任一互動按鈕]
C --> F[點歷史查看]
E --> G[提示請先登入]
F --> G
D --> H[Google 登入]
G --> H
H --> I[登入成功回首頁]
B -- 是 --> J[首頁可操作]
I --> J
```

## 1.1 首頁統計類型切換流程

```mermaid
flowchart TD
A[首頁甜甜圈圖與類別摘要] --> B{切換統計類型}
B -- 支出類別 --> C[顯示本月支出類別占比]
B -- 收入類別 --> D[顯示本月收入類別占比]
B -- 總現金流 --> E[顯示本月總收入與總支出占比]
E --> F[中心顯示收入扣除支出後的淨現金流]
```

## 2. 首頁批次新增流程

```mermaid
flowchart TD
A[首頁左側批次新增] --> B[預設支出與今天日期，輸入多筆帳目]
B --> C{是否每列必填皆完成}
C -- 否 --> D[不可送出]
C -- 是 --> E[送出整批]
E --> F{是否成功}
F -- 否 --> G[顯示失敗原因並保留內容]
F -- 是 --> H[顯示成功訊息]
H --> I[清空表單]
I --> J[更新首頁甜甜圈圖、類別摘要列表、今日明細]
```

## 3. 首頁今日明細流程

```mermaid
flowchart TD
A[首頁今日明細預設收合] --> B[點查看]
B --> C[顯示今天帳目，預設 5 筆]
C --> D[依類別排序，可切換每頁 10 / 15]
C --> E[點查看全部]
E --> F[進入歷史查看頁]
F --> G[沿用目前歷史查看條件]
```

## 4. 歷史查看主要流程

```mermaid
flowchart TD
A[進入歷史查看頁] --> B[套用預設條件]
B --> C[日期區間：本月第一天至今天]
B --> D[類型：支出類別]
B --> E[趨勢圖：當年度每月總現金流與累積總現金流]
C --> F[左側明細載入]
D --> G[右上甜甜圈圖載入]
E --> H[右下折線圖載入]
F --> I[全頁同步顯示]
G --> I
H --> I
```

## 7. 歷史查看明細操作流程

```mermaid
flowchart TD
A[歷史查看左側明細預設收合] --> B[點查看]
```

## 8. 帳目編輯共用流程

適用範圍：歷史查看明細。

```mermaid
flowchart TD
A[點擊明細編輯] --> B[開啟編輯彈窗]
B --> C[修改可編輯欄位]
C --> D{按確定或取消}
D -- 取消 --> E[關閉彈窗不變更資料]
D -- 確定 --> F[送出更新]
F --> G{是否成功}
G -- 否 --> H[顯示失敗原因並保留當前內容]
G -- 是 --> I[顯示更新成功提示]
I --> J[更新目前所在頁面的列表與圖表]
```

## 9. 帳目刪除共用流程

適用範圍：歷史查看明細。

```mermaid
flowchart TD
A[點擊明細刪除] --> B[開啟刪除確認視窗]
B --> C[顯示日期、類別、金額]
C --> D{是否確認刪除}
D -- 否 --> E[關閉視窗不刪除資料]
D -- 是 --> F[刪除資料]
F --> G{是否成功}
G -- 否 --> H[顯示失敗原因並不更新資料]
G -- 是 --> I[顯示刪除成功提示]
I --> J[更新目前所在頁面的列表與圖表]
J --> K{是否在歷史查看且當前頁已無資料}
K -- 否 --> L[留在目前頁面狀態]
K -- 是 --> M[自動回上一頁]
M --> N{上一頁是否仍無資料}
N -- 否 --> O[顯示上一頁資料]
N -- 是 --> P[顯示空狀態]
```
