# 線框圖 V2

## 1. 首頁

```text
+----------------------------------------------------------------------------------+
| 個人記帳                              首頁 | 歷史查看 | 登入 / 使用者資訊          |
+----------------------------------------------------------------------------------+
| 批次新增表單                            | 本月甜甜圈圖與摘要                    |
|-------------------------------------------|----------------------------------------|
| [支出] [今天日期] [金額] [類別] [備註]    | [支出] [收入] [總現金流]              |
|                                           |                                        |
| [新增一列] [送出]                         |              (甜甜圈圖)                |
|                                           |                                        |
|                                           | [查看明細] 類別摘要                    |
|                                           | 類別 | 金額 | 占比                    |
|                                           |----------------------------------------|
|                                           | 今日明細                               |
|                                           |----------------------------------------|
|                                           | 今日明細 [查看]                        |
|                                           | 展開後：今日明細 [5 / 10 / 15] [收合] |
|                                           | 日期 | 類型 | 類別 | 金額 | 備註     |
|                                           | 建立時間                               |
|                                           | [查看全部]                            |
+----------------------------------------------------------------------------------+
```

首頁規則：

- 未登入使用者可以看到版型，但不可使用受保護操作。
- 已登入使用者可以新增帳目並查看自己的資料。
- 今日明細預設收合，展開後才顯示列表、筆數選擇與查看全部。
- 甜甜圈圖與類別摘要使用相同的本月資料來源，類別摘要預設收合；圖例標籤需保留固定間距。
- Favicon 使用符合頁面柔和風格的可愛金錢符號。

## 2. 歷史查看

```text
+----------------------------------------------------------------------------------+
| 個人記帳                              首頁 | 歷史查看 | 使用者資訊                 |
+----------------------------------------------------------------------------------+
| 查詢與明細列表                          | 日期區間甜甜圈圖                      |
|-------------------------------------------|----------------------------------------|
| 日期：[開始] ~ [結束]                     | [支出] [收入] [總現金流]              |
| 明細類型：[支出 / 收入]                   |                                        |
|                                           |              (甜甜圈圖)                |
| 支出或收入明細                            |                                        |
| 日期 | 類型 | 類別 | 金額 | 備註         |----------------------------------------|
| ... [編輯] [刪除]                         | 年度現金流趨勢                         |
| ... [編輯] [刪除]                         |----------------------------------------|
|                                           | 每月現金流 / 累積現金流                |
| [10 / 15 / 20] [上一頁] [下一頁]          |              (折線與長條圖)            |
+----------------------------------------------------------------------------------+
```

歷史查看規則：

- 日期區間共用於明細列表、甜甜圈圖與年度趨勢年份。
- 明細類型與甜甜圈圖模式可獨立控制。
- 編輯使用彈窗。
- 刪除使用確認彈窗。
- loading 狀態應輕量且靠近受影響區塊。
- 各區塊無資料時顯示空狀態。

## 3. 編輯彈窗

```text
+--------------------------------------+
| 編輯帳目                              |
+--------------------------------------+
| 類型：    [選擇]                       |
| 日期：    [日期]                       |
| 金額：    [輸入]                       |
| 類別：    [選擇或自訂輸入]             |
| 備註：    [輸入]                       |
|                                      |
|                 [取消] [確認]          |
+--------------------------------------+
```

## 4. 刪除確認

```text
+--------------------------------------+
| 確認刪除                              |
+--------------------------------------+
| 日期：   <帳目日期>                    |
| 類別：   <類別>                        |
| 金額：   <金額>                        |
|                                      |
| 確定要刪除此筆帳目嗎？                 |
|                                      |
|                 [取消] [刪除]          |
+--------------------------------------+
```

## 5. 響應式規則

- 桌機版維持上述左右雙欄結構。
- 手機版可將區塊改為上下排列。
- 手機版必須保留所有必要欄位與操作。
- 小尺寸畫面下，表格可改為卡片式列表，但必要欄位仍需可見。
- 圖表可縮放，但不得改變資料計算規則。
