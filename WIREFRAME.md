# WIREFRAME.md

## 1. 首頁線框

```text
+--------------------------------------------------------------------------------------+
| Logo / 產品名稱                           首頁 | 歷史查看 | [登入 / 使用者資訊]      |
+--------------------------------------------------------------------------------------+
| 左側：批次新增區                           | 右上：甜甜圈圖與類別摘要                   |
|-------------------------------------------|-------------------------------------------|
| [收入/支出] [日期] [金額] [類別] [備註]    | [支出類別 / 收入類別 / 總現金流]           |
|                                           |                                           |
| [繼續新增下一筆]                           |                (甜甜圈圖)                  |
| [送出]                                    |      hover: 收入/支出 類別 金額 占比       |
|                                           |                                           |
|                                           | 類別摘要列表                               |
|                                           | 類別 | 金額 | 占比                         |
|                                           | ...                                       |
|                                           |-------------------------------------------|
|                                           | 右下：最近明細                             |
|                                           |-------------------------------------------|
|                                           | 日期 | 收支 | 類別 | 金額 | 備註 | 建立日期 |
|                                           | ...                                       |
|                                           | [每頁 5 / 10 / 15]                        |
|                                           | [查看更多]                                 |
+--------------------------------------------------------------------------------------+
```

### 首頁說明
- 未登入時：
  - 可看到整體版型
  - 不顯示真資料
  - 點互動按鈕時提示先登入
- 歷史查看在未登入時不可進入
- 登入後右上顯示頭像、帳號名稱、登出
- 右上區塊包含甜甜圈圖與類別摘要列表。
- 甜甜圈圖與類別摘要可切換支出類別、收入類別與總現金流。
- 總現金流模式顯示本月總收入與總支出占比，中心金額顯示本月淨現金流。
- 類別摘要列表顯示欄位：
  - 類別
  - 金額
  - 占比

## 2. 歷史查看頁線框

```text
+--------------------------------------------------------------------------------------+
| Logo / 產品名稱                           首頁 | 歷史查看 | [使用者資訊]             |
+--------------------------------------------------------------------------------------+
| 左側：查詢 + 明細列表                      | 右側上：甜甜圈圖                           |
|-------------------------------------------|-------------------------------------------|
| 日期： [起日] ~ [迄日] / [單日]            | [支出類別 / 收入類別 / 總現金流]           |
| [回到預設]                                 |                                           |
|                                           |                (甜甜圈圖)                  |
| 支出明細 / 收入明細                        |      hover: 收入/支出 類別 金額 占比       |
| 日期 | 收支 | 類別 | 金額 | 備註           |                                           |
| ... [編輯] [刪除]                          |-------------------------------------------|
| ... [編輯] [刪除]                          | 右側下：折線圖                             |
|                                           |-------------------------------------------|
| [每頁 10 / 15 / 20]                        | [每月總現金流 / 累積總現金流]              |
| [上一頁] [下一頁]                          |                                           |
|                                           |                 (折線圖)                  |
|                                           |   hover: 月份 / 模式 / 金額                |
+--------------------------------------------------------------------------------------+
```

### 歷史查看頁說明
- 左右共用同一組日期條件
- 類型切換放在右上圖表區上方，並同步影響左側明細與甜甜圈圖
- 總現金流模式顯示查詢區間內總收入與總支出占比，中心金額顯示淨現金流
- 右下折線圖顯示查詢起始日期所在年份的每月總現金流與累積總現金流
- 左側明細可直接編輯 / 刪除
- 編輯採彈窗
- 刪除採確認視窗
- 資料同步更新時只顯示非常輕量 loading
- 單日查詢時：
  - 甜甜圈圖照常顯示
  - 折線圖照常顯示該日所在年份的年度現金流趨勢
- 無資料時：
  - 左側、甜甜圈圖、折線圖都顯示空狀態

## 3. 編輯彈窗線框

```text
+--------------------------------------+
| 編輯帳目                              |
+--------------------------------------+
| 收入/支出: [下拉]                     |
| 日期:       [日期]                    |
| 金額:       [輸入框]                  |
| 類別:       [下拉]                    |
| 備註:       [輸入框]                  |
|                                      |
|                 [取消] [確定]         |
+--------------------------------------+
```

### 編輯彈窗說明
- 可修改：
  - 收入 / 支出
  - 日期
  - 金額
  - 類別
  - 備註
- 不可修改：
  - 建立日期
- 成功後顯示更新成功並自動消失
- 失敗時顯示失敗原因

## 4. 刪除確認彈窗線框

```text
+--------------------------------------+
| 確認刪除                              |
+--------------------------------------+
| 日期：2026-04-01                      |
| 類別：飲食                            |
| 金額：300                             |
|                                      |
| 確定要刪除此筆資料嗎？                |
|                                      |
|                 [取消] [確認刪除]     |
+--------------------------------------+
```

### 刪除彈窗說明
- 顯示日期、類別、金額
- 成功後更新目前所在頁面的相關列表與圖表
- 若在歷史查看頁刪除後當前頁無資料，自動回上一頁
